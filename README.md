## How to Use
1. Go to the Data Upload page
2. Upload your CSV files
3. Map the columns if needed
4. Navigate to Dashboard for insights
5. Use Forecasting for inventory planning

## Data Format
- Sales CSV: Should contain date, style ID, and quantity columns
- Returns CSV: Should contain return date and style ID
- Catalog CSV: Style listing data
